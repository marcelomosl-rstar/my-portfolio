Voxel Engine

Voxel Engine is my custom-built engine designed for rendering and manipulating 3D voxel-based worlds using OpenGL and C++. It offers a robust framework for creating dynamic, block-based environments with efficient rendering and real-time updates. Currently a work in progress, the engine focuses on flexibility and performance for voxel-based game development.
